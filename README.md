Title:
Library Management System
Description: 
A simple library management system that allow user to add books, borrow books, search books, return books, show all books and total borrowed books
Technologies Used:
1.Python
2.os module
3.json file handling
How to Run:
Install pyhton on your editor
Run:Main(1).py and ensure that library.json is in the same folder.
